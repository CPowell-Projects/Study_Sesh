# Study_Sesh
Study_Sesh is a simple student assignment tracker built with Flask.

## Features
Add assignments with course name and due date
Prevent invalid (past) due dates
Automatically sort assignments by soonest due date
Delete assignment
Persistent storage using JSON
Server-side validation and error handling

## Tech Stack
Python
Flask
HTML
JSON

## How to run
pip install -3 requirements.txt
python app.py
Then open the url (should be http://127.0.0.1:5000)
